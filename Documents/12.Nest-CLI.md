# CLI Nest

Sử dụng lệnh này để xem các gợi ý cli nest
```bash
npx nest --help
```

Thứ tự khi tạo một module mới:
1. Generate module
```bash
npx nest generate mo routes/profile
```

2. Generate controller
```bash
npx nest generate co routes/profile
```

3. Generate service
```bash
npx nest generate s routes/profile
```
